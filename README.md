# 🤖 Честные отзывы Казани - Telegram Mini App

Telegram Mini App для сбора, хранения и публикации реальных отзывов пользователей о заведениях города Казань.

## 🎯 Особенности

- **📱 Telegram Mini App** — полноценный интерфейс внутри Telegram
- **🔐 Интеграция с Telegram** — авторизация через Telegram ID
- **🎨 Адаптивный дизайн** — использует тему Telegram (светлая/тёмная)
- **⚡ Быстрая работа** — Next.js + React 19

## 📋 Возможности

### Для пользователей:
- ✍️ **Оставить отзыв** — выбор заведения из списка + оценки (общая + 4 детальных) + текст + фото
- 🔎 **Найти отзывы** — поиск по категории с фильтрами
- 🏆 **Рейтинг заведений** — ТОП и худшие места Казани
- 📍 **Найти рядом** — геопоиск заведений рядом с пользователем
- 🧑‍💻 **Профиль** — статистика пользователя, подписки на категории
- 👍👎 **Голосование** — оценка полезности отзывов
- 🚩 **Жалобы** — сообщить о нарушении

### Для администраторов:
- 🏪 **Добавление заведений** — через команду `/addplace` или веб-панель
- 📝 **Модерация** — одобрение/отклонение/редактирование отзывов
- 👥 **Управление пользователями** — блокировка/разблокировка
- 📊 **Статистика** — аналитика по проекту
- 📢 **Рассылка** — отправка сообщений пользователям

### Защита от накрутки:
- 1 отзыв на одно заведение от одного пользователя
- Модерация всех отзывов перед публикацией
- Система жалоб на недостоверные отзывы

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
bun install
```

### 2. Настройка окружения

Скопируйте `.env.example` в `.env.local` и заполните переменные:

```env
# Telegram Bot Token (получить у @BotFather)
BOT_TOKEN=your_telegram_bot_token_here

# Канал для публикации отзывов
CHANNEL_ID=@your_channel_username

# Mini App URL (обязательно для Web App)
WEBAPP_URL=https://your-domain.com

# ID администраторов (через запятую)
ADMIN_IDS=123456789,987654321

# База данных
DATABASE_URL=file:./db/custom.db
```

### 3. Инициализация базы данных

```bash
bun run db:push
```

### 4. Запуск в режиме разработки

```bash
# Запуск веб-приложения (Mini App)
bun run dev

# Запуск бота (в другом терминале)
bun run bot
```

### 5. Настройка Mini App в BotFather

1. Откройте @BotFather в Telegram
2. Выберите вашего бота
3. Нажмите "Bot Settings" → "Menu Button"
4. Выберите "Web App"
5. Введите URL вашего Mini App (из WEBAPP_URL)

Теперь при нажатии на кнопку меню в боте откроется Mini App!

### 6. Для локальной разработки

Для тестирования Mini App локально нужен туннель (например, ngrok):

```bash
# Установите ngrok, затем:
ngrok http 3000

# Добавьте полученный URL в .env.local:
WEBAPP_URL=https://xxxx.ngrok.io
```

## 📝 Процесс работы Mini App

### Шаг 1: Администратор добавляет заведения

**ВАЖНО:** Пользователи НЕ вводят названия вручную. Администратор заранее добавляет все заведения в базу.

Способы добавления:
1. **Команда `/addplace`** в боте — интерактивная форма
2. **Веб-панель** — через API

### Шаг 2: Пользователь открывает Mini App

1. Нажимает кнопку меню в боте → открывается Mini App
2. Выбирает категорию на главном экране
3. Выбирает заведение из списка
4. Читает отзывы или нажимает "Написать отзыв"

### Шаг 3: Написание отзыва в Mini App

1. Выбор заведения из списка (добавленные админом)
2. Оценка слайдерами (1-10):
   - ⭐ Общая оценка
   - 🍽 Еда
   - 🤝 Сервис
   - 🏠 Атмосфера
   - 💰 Цена/качество
3. Текст отзыва (минимум 20 символов)
4. Отправка на модерацию

## 🚀 Деплой

### Переменные окружения для продакшена:

```env

BOT_TOKEN=your_bot_token
CHANNEL_ID=@your_channel
ADMIN_IDS=123456789
WEBHOOK_URL=https://your-domain.com/api/bot
```

### Webhook (рекомендуется для продакшена):

```typescript
// Установка webhook

```

### Polling (для разработки):

```typescript

```
